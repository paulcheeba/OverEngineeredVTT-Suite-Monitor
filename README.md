# OverEngineeredVTT-Suite-Monitor
A suite monitor for OverEngineeredVTT module updates and reminders.

## Foundry VTT

- Requires Foundry VTT v13+
- Entry point: scripts/main.js

## Watch List

Edit the in-code watch list in scripts/main.js (WATCHED_MODULES). This module only checks watched modules which are installed and enabled in the world.
