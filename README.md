# OverEngineeredVTT-Suite-Monitor

A suite monitor for OverEngineeredVTT module updates and reminders.

## Foundry VTT

- Requires Foundry VTT v13+

## Watch List

This module tracks the following OverEngineeredVTT modules:

- [About Time Next](https://github.com/paulcheeba/about-time-next)
- [Chat Pruner](https://github.com/paulcheeba/chat-pruner)
- [Find and Replace](https://github.com/paulcheeba/find-and-replace)
- [OEV Suite Monitor](https://github.com/paulcheeba/OverEngineeredVTT-Suite-Monitor) (self-monitoring)
- [Window Controls Next](https://github.com/paulcheeba/window-controls-next)

The module checks for updates to installed modules (both enabled and disabled in Testing Mode, or only enabled modules in normal operation).
